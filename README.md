# Titanic Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project demonstrates Exploratory Data Analysis (EDA) on the Titanic dataset using the Pandas library in Python. The objective is to explore the dataset, identify missing values, summarize statistical information, and extract meaningful insights.

## 🎯 Objectives

- Load the Titanic dataset
- Explore the dataset structure
- Generate descriptive statistics
- Identify missing values
- Perform filtering and slicing
- Group and aggregate data
- Analyze categorical data
- Compute feature correlations
- Document key observations

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas

## 📂 Dataset

- Titanic Dataset

## 📖 Concepts Covered

- `pd.read_csv()`
- `df.info()`
- `df.describe()`
- `isnull()`
- `dropna()`
- Filtering
- Slicing
- `groupby()`
- `agg()`
- `value_counts()`
- `df.corr()`

## 📊 Key Findings

- The dataset contains 891 passenger records and 12 features.
- Missing values were identified in the Age, Cabin, and Embarked columns.
- More passengers did not survive than survived.
- Passenger characteristics varied across different groups.
- Correlation analysis revealed relationships among numerical features.
